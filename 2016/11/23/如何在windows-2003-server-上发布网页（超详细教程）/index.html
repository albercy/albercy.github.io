如何在windows 2003 server 上发布网页（超详细教程）

发表于 2016-11-23

名词解释(转自花生壳)

内网：就像你小时候，什么事都有爸妈顶着，爸妈做主，都要通过爸妈。没有身份证。

  • 内网保留地址有如下3种形式：
    10.x.x.x
    172.16.x.x至172.31.x.x
    192.168.x.x

外网：就像你长大了，什么事都可以自己做主了。有身份证了，要自己承担法律责任了。

内网也就是局域网，最直观的就是像网吧，公司内部的电脑，都是用交换机，HUB，路由连起来的，然后通过光钎和猫接入INTERNET的。

外网就像你家里的一台电脑。用猫拨号上的，有个外部IP的。

为什么要说这两个名词呢，就是希望大家不要混淆ip地址或者以为ip地址只有一种，因为这两个概念挺重要的。

前期准备

windows 2003的服务器（因为教程是在windows 2003下进行的）、网站文件、iis管理工具,路由和远程访问（这两个都是系统自带的）

ps:正常来说，以上这些（除了iis）都是不需要我们去下载配置的

iis安装教程：

  1. 远程连接到你的服务器（此时所输入的就是你的公网ip）
  2. 进了服务器后，打开控制面板里的 “添加或删除程序”

  1. 点击 “添加/删除 windows组件（A）”，然后点击 “应用程序服务器” ，然后点击右下方的详细信息,勾选 “Internet 信息服务（IIS）”

（如果有其他需要可勾选其他项，如ASP.NET，另如果iis的配置也可在iis详细信息里添加）

  1. 完成以上工作后一直点确定就可以完成安装了，这里就不演示了（过程中可能需要的文件参考附注）

 附：iis 安装过程可能需要的文件 百度网盘

发布网站

  1. 打开iis管理器
